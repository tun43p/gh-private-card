## Getting started

### Check project

```sh
cargo check # cargo c
```

### Run project

```sh
cargo run # cargo r
```

#### Run and watch project

_If you don't have `cargo-watch` installed: `cargo install cargo-watch`_

```sh
cargo watch -x run 
```
